amux vs Goose

Both open source. Goose is a single agent; amux manages a fleet.

Goose (by Block) is an open-source AI coding agent with MCP tool support. amux manages many Claude Code sessions in parallel with orchestration.

Key differences

FeatureamuxGoose
Agent countDozens in parallelSingle agent
Self-healingAuto-compact + restartManual recovery
DashboardWeb UI with live monitoringCLI only
CoordinationKanban board + REST APINo multi-agent coordination
Tool systemClaude Code built-in toolsMCP protocol
LLM supportClaude (via Claude Code)Multi-provider

Get started with amux

Run dozens of Claude Code agents in parallel. Python 3 + tmux. Open source.

git clone https://github.com/mixpeek/amux && cd amux && ./install.sh
amux register myproject --dir ~/Dev/myproject --yolo
amux start myproject
amux serve  # → https://localhost:8822
