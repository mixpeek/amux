amux — Your AI engineering team

What it replaces: Junior engineers, freelancers, or contractors you can't afford. amux lets you run 10–50 Claude Code sessions simultaneously, each working on a different task. Queue a backlog before bed; wake up to PRs.

Cost: Free and open source. You pay only for Claude/Codex API tokens — typically $5–30 per overnight run depending on task complexity.

Time saved: If parallel agents complete 20 tasks overnight that would have taken you 40 hours, that's 40 hours you spend on product decisions, user research, or sleep. The leverage is qualitative, not just quantitative.

Setup: 15 minutes. github.com/mixpeek/amux

Claude (Anthropic)

What it replaces: Research assistants, technical writers, copywriters, and analysts. Claude is the most capable general-purpose AI for complex reasoning tasks in 2026. Use the API for programmatic access via amux sessions, or Claude.ai Pro for interactive work.

Cost: Claude Pro: $20/month. API: ~$3–15 per million tokens depending on model. Most solo use cases cost $10–50/month on the API.

Cursor

What it replaces: Hours of manual coding, Stack Overflow lookup, and boilerplate writing. The best AI-augmented IDE for interactive development — use alongside amux for the sessions you're actively in vs. the ones running overnight.

Cost: $20/month Pro. Free tier available. How amux and Cursor complement each other →

Notion AI

What it replaces: Manually writing documentation, meeting summaries, project specs, and SOPs. If you're already in Notion, the AI add-on is high-ROI — it drafts directly in your workspace without context switching.

Cost: $10/month add-on on Plus plans and above.

Perplexity

What it replaces: Hours of Googling for competitive analysis, technical research, and market sizing. Perplexity returns cited, synthesized answers rather than links — dramatically faster for research tasks. Pro unlocks Claude and GPT-4 for better reasoning.

Cost: Free tier available. Pro: $20/month.

Make (Integromat)

What it replaces: Manual repetitive workflows — syncing data between tools, triggering notifications, processing form submissions, sending follow-up emails. Make's visual builder requires no code for most automations.

Cost: Free tier (1000 ops/month). Basic: $10/month. Standard: $16/month.

v0 (Vercel)

What it replaces: Days of UI design and frontend scaffolding. Describe a component or page in natural language; v0 generates production-ready React + Tailwind. Best for solopreneurs who can code but want to skip the design/scaffolding phase.

Cost: Free tier (limited generations). $20/month for unlimited.

Bolt.new

What it replaces: Days or weeks of MVP scaffolding. Describe an app; Bolt generates a working full-stack application in the browser. Best for validating ideas before investing in proper development. amux vs Bolt.new →

Cost: Free tier. $20/month Pro.

Linear AI

What it replaces: Manual sprint planning, issue triage, and project status reporting. Linear's AI features summarize issue context, suggest priorities, and draft issue descriptions. Best project management tool for solo technical founders.

Cost: $8/member/month (Starter). Free for personal use.

The stack recommendation

For a solo developer focused on shipping code: amux + Claude API + Cursor is the core. amux handles overnight parallel work; Cursor handles interactive development; Claude API handles analysis and writing. Add Perplexity and Make for research and automation. Budget: $70–120/month for the full stack. Output: comparable to a 3-person team.
