Features

Everything amux adds on top of Claude Code.

Core features

  • Web Dashboard -- session cards, peek mode, workspace, board, and reports in a single-page PWA.
  • Self-Healing Watchdog -- auto-compacts on context overflow, restarts on corruption, replays last message.
  • Remote Control iOS App & PWA — native iOS app + PWA. Monitor sessions, approve actions, steer agents from anywhere.
  • Agent Coordination -- REST API orchestration with atomic task claiming via SQLite CAS.
  • Single-File Architecture -- one Python file, zero dependencies beyond python3 and tmux.
  • Cloud Tunnel -- stable public HTTPS URL for your local dashboard. No port forwarding required. Requires amux Cloud ($20/mo).

Get started with amux

Run dozens of Claude Code agents in parallel. Python 3 + tmux. Open source.

git clone https://github.com/mixpeek/amux && cd amux && ./install.sh
