Mobile-First Agent Management

Your agents work 24/7. You should be able to check on them from anywhere.

amux is the remote control for your AI engineering team. Think of it like GitHub Mobile or Tailscale — the work happens on your machine; amux puts the dashboard in your pocket. Monitor live Claude Code sessions, approve pending actions, peek at terminal output, steer agents, and recover problems from your iPhone or Android — without SSH, without going back to your desk.

Two ways to access it: install the native iOS app from the App Store, or open your amux dashboard URL in Safari/Chrome and add it to your home screen as a PWA. Offline support means queued commands replay when you reconnect.

What you can do from your phone

  • Check overnight agent runs before getting out of bed
  • Approve permission prompts so an agent doesn't stall for hours
  • See exactly why an agent stopped
  • Assign a new task to an idle session
  • Peek into live terminal output
  • Keep coding while commuting or traveling

Get started with amux

Run dozens of Claude Code agents in parallel. Python 3 + tmux. Open source.

git clone https://github.com/mixpeek/amux && cd amux && ./install.sh
amux register myproject --dir ~/Dev/myproject --yolo
amux start myproject
amux serve  # → https://localhost:8822
