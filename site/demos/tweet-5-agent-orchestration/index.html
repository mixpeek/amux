Agents That Coordinate Themselves

Agents discover peers and delegate work on their own.

Agents talk to each other too. Every session gets environment variables and a shared memory file with the full REST API. Agents discover peers and delegate work on their own. Task claiming is atomic via SQLite CAS — dozens of agents race for the same queue, zero duplicated work, no Redis.

  • $AMUX_SESSION and $AMUX_URL injected into every session at startup
  • Shared global memory gives every agent the full REST API reference
  • Agents discover peers and delegate work — no hardcoded routing
  • Atomic task claiming: SQLite CAS — no Redis, no lock broker, no duplicates
