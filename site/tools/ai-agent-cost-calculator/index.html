// How to cut costs
Save 50-65%

Model routing

Use Sonnet for file reads, boilerplate, and simple edits. Reserve Opus for architecture decisions and complex debugging. Most teams do 80/20 Sonnet/Opus.

Save 20-40%

Context discipline

Write focused CLAUDE.md files. Use .claudeignore to exclude generated files. Smaller context = fewer tokens re-read after compaction.

Save 10-25%

Monitor & kill

Track per-session spend in real time. Kill agents stuck in loops before they burn through your budget. Set daily spend alerts.

Track real costs, not estimates

This calculator gives you a projection. amux gives you the actual numbers — per-session token spend, daily cost breakdowns, and alerts when agents burn through your budget. Free, open-source, self-hosted.

# Install and see real spend in 2 minutes
git clone https://github.com/mixpeek/amux && cd amux && ./install.sh
amux register myproject --dir ~/Dev/myproject
amux serve  # → https://localhost:8822
