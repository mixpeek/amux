Glossary

Key concepts in AI agent orchestration and parallel development.

  • Agent Multiplexer
    Agent Multiplexer: A tool that runs and manages multiple AI coding agents simultaneously.
  • Self-Healing AI Agent
    Self-Healing AI Agent: An AI agent that automatically recovers from crashes, context exhaustion, and stuck states.
  • Context Compaction
    Context Compaction: The process of summarizing conversation history to free up context window space in an AI agent.
  • YOLO Mode
    YOLO Mode: A configuration that auto-approves tool-use prompts in Claude Code for fully unattended operation.
  • Agent Orchestration
    Agent Orchestration: Coordinating multiple AI agents to work together on complex tasks.
  • Parallel Coding Agents
    Parallel Coding Agents: Multiple AI agents working on different coding tasks at the same time.
  • Token Accounting
    Token Accounting: Tracking token usage and costs across multiple AI coding sessions.
  • Atomic Task Claiming
    Atomic Task Claiming: A concurrent-safe mechanism for agents to claim tasks without duplicating work.
  • Session Persistence
    Session Persistence: The ability for AI agent sessions to survive restarts and maintain their identity.
  • Prompt Replay
    Prompt Replay: Automatically re-sending the last message after an AI agent crash to recover progress.
  • Global Memory
    Global Memory: A shared knowledge base accessible to all AI agents in a multiplexer.
  • Background Watchdog
    Background Watchdog: A monitoring process that detects and recovers from AI agent failures automatically.
  • Conversation Fork
    Conversation Fork: Cloning an AI session's history into multiple new sessions for parallel exploration.
  • Terminal Peek
    Terminal Peek: Viewing an AI agent's live terminal output without attaching to the session.
  • PWA (Progressive Web App)
    PWA (Progressive Web App): A web application installable on mobile devices with offline support.
  • tmux Session Management
    tmux Session Management: Using tmux terminal multiplexer to run persistent background processes.
  • REST API Orchestration
    REST API Orchestration: Using HTTP endpoints to coordinate AI agent communication and task management.
  • iCal Sync
    iCal Sync: Exporting task deadlines as calendar events via the iCal format.
  • SQLite-Backed Board
    SQLite-Backed Board: A kanban board using SQLite for concurrent-safe task management.
  • Git Worktree Isolation
    Git Worktree Isolation: Using git worktrees to give each AI agent its own working copy of a repository.
  • Headless Coding Agent
    A coding agent that runs without a GUI, controlled entirely via CLI or API.
  • Vibe Coding
    A programming style where developers describe intent in natural language and AI generates the implementation.
  • Agent Fleet
    A group of AI coding agents running in parallel, coordinated toward a shared goal.
  • LLM Context Window
    The maximum amount of text an LLM can process in a single request — a key constraint in long-running agent sessions.
  • Unattended Coding
    Running AI coding agents without human supervision — enabled by self-healing, YOLO mode, and atomic task management.
